MCStatusWidget is a widget which show INFO Of Bukkit server.

== Description ==

It show:
HostName, IP:Port, Version, World, Player online / Player max, Player list, Plugin list.
Dont need MineQuery Plugin.
Need only enable-query=true line in server.properties.

== Installation ==
1. Upload `MCStatusWidget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place MCStatusWidget into sidebar. Write down the IP and Port and Save.
4. Add enable-query=true Line to server.properties (If you dont have one)
